Description détaillée

Fenêtre pour l'import de nouvelles données.

Auteur
Julien Richard

Cette fenêtre permet à un utlisateur d'importer de nouvelles données dans le but de créer une nouvelle ville en 3D. 6 boutons sont présents pour 6 données différentes : le bâti, les routes, les arbres, les zones d'herbes, les surface d'eaux, et la topographie. Pour chacune des catégories de donnée un bouton d'aide est disponible pour expliquer ce que le logiciel neccessite et aussi comment se procurer cette donnée.
